# Smart Home Dashboard

Веб-интерфейс для управления устройствами умного дома, отображения состояния датчиков и работы с ИК-командами.

## Возможности

- просмотр устройств и их состояния
- управление устройствами и сценариями
- отображение истории показаний
- работа с ИК-пультами и командами

## Запуск

```bash
npm install
npm run dev
```
